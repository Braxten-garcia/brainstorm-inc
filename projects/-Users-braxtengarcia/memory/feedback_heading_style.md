---
name: Heading style — sentence case
description: All headings in blog posts and marketing content must be sentence case
type: feedback
---

All headings (H1, H2, H3) must be written in sentence case: only the first word and proper nouns are capitalized.

**Why:** User preference, explicitly requested.

**How to apply:** Applies to all blog copy, event materials, and any written content with headings. Example: "How to use OneNote for organization" not "How to Use OneNote for Organization". Proper nouns (OneNote, Microsoft, Copilot, BrainStorm) stay capitalized.
